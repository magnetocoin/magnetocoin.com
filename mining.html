Mine Some MGT Yourself

Currently The MGT Network Difficulty allows for Feasible solo mining. Getting Started is Easy. And you will greatly help in securing the magnetocoin network from malicious actors and earn great many magnetos for doing so as rewards. To be an early bird and get started with solo mining

  1. Start Your Core Wallet Client
  2. Click on Help Menu
  3. Click on "Debug Window"
  4. Go to console Tab
  5. Give the command setgenerate true and press Return
  6. Happy Mining
  7. To stop mining use the command setgenerate false and press return
